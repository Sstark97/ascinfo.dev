![Chat Now Header](/images/projects/chat-now/chat-now-banner.webp)

La aplicación consiste en una aplicación web de mensajería instantánea en tiempo real, en la que un usuario se podrá registrar, iniciar sesión, agregar contactos y mandar mensajes a los mismos. Además es una aplicación web progresiva, por lo que se puede instalar en cualquier dispositivo y utilizarla como una aplicación nativa.

## Tecnologías
- React
- TypeScript
- TailwindCSS
- Prisma
- NextJS
- NextAuth
- NextPWA
- Socket.io
